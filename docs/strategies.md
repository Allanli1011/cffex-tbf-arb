# 套利策略详解

> 本文件为每类策略的公式、信号、风险点的工作手册，开发时直接引用。

## 1. 基差套利

### 关键变量

| 符号 | 含义 |
|------|------|
| `P_clean` | 现券净价 |
| `AI` | 应计利息 |
| `P_dirty = P_clean + AI` | 现券全价 |
| `F` | 期货价格 |
| `CF` | 转换因子 |
| `Carry` | 持有收益（票息收入 - 资金占用成本） |

### 公式

```
Invoice Price        = F × CF + AI_T
Gross Basis          = P_clean - F × CF
Net Basis (BNOC)     = Gross Basis - Carry
IRR                  = [(F × CF + AI_T) / (P_clean + AI_0) - 1] × 365 / n
```

### CTD 判定
- 主流方法：在交割日按 IRR 最大的券作为 CTD
- 简化方法：在远离交割日时，使用 Net Basis 最小的券近似
- 切换风险：当某券与 CTD 的 IRR 差距 < 5bp 且久期相近时，标记 "高切换概率"

### 信号阈值（可在 configs 中调整）
- IRR - R007 > 30bp → 正向基差信号
- IRR - R007 < -20bp → 反向基差信号
- Net Basis 分位数 > 90% 或 < 10% → 极值告警

### 风险点
- 国内融券难，反向套利落地受限
- 交割期权价值在临近交割时显著上升，IRR 失真
- CF 取整误差对短券影响放大

## 2. 跨期套利

### 公式

```
Spread       = F_far - F_near
Theoretical  ≈ F_near × (r_implied - c_carry) × (T_far - T_near) / 365
Z-score      = (Spread - μ_window) / σ_window
```

### 信号
- |Z| > 2 且距交割 > 20 个交易日 → 入场候选
- 移仓窗口（交割前 10-15 个交易日）期间，主力持仓迁移率 > 50% 时关注 roll yield

### 风险点
- 移仓拥挤导致价差短期偏离均值
- 临近交割时近月波动放大
- 跨期 PnL 高度依赖资金成本假设

## 3. 收益率曲线套利

### DV01 中性比例

```
notional_i = (target_DV01) / DV01_per_contract_i
```

蝶式策略（2-5-10）权重示例：

```
设 DV01_TS = a, DV01_TF = b, DV01_T = c
+1 张 TS, -w_TF 张 TF, +w_T 张 T
约束：a - w_TF × b + w_T × c = 0  (DV01 中性)
      凸性约束（可选，二阶矩中性）
```

### 信号
- 蝶式价差 Z-score > 2 → 做空蝶式
- 期货隐含曲线与现券曲线背离 > 10bp → 套利机会

### 风险点
- 各品种流动性差异，TL 与 TS 流动性远不及 T
- 跨品种保证金占用高
- 曲线形态在政策事件下可能大幅平移

## 4. 拓展策略

### 4.1 期货 vs IRS
- 用期货合成隐含 yield，对比 IRS 同期限利率
- 注意：期货标的为国债，IRS 多为 FR007 浮动腿，存在信用 / 基准利差

### 4.2 国债 vs 国开利差
- 用 T 合约 + 现券国开债构造久期中性利差头寸
- 历史分位数监控国开-国债利差

### 4.3 跨市场套利（银行间 vs 交易所）
- 同券在银行间和交易所价差
- 资金跨市场效率影响套利空间

### 4.4 期货+回购合成短端利率
- 通过基差套利组合合成短期融资利率
- 与实际 R007 比较，识别融资套利机会
